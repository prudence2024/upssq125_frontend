API(Application Programming Interfaces)

API(Application Programming Interfaces) is like a menu at a restaurant, it list all the dishes(data or services) that you can order(request). You dont need to know how the kitchen(server) makes the food , you just need to know what to ask for and what you will get back and how to ask for it.

Understanding Ajax (Asynchronous Javascript and Xml):

Web History API

The Web History API provides easy methods to access the windows.history object.The window.history object contains the URLs (Web Sites) visited by the user.

The History back() Method

The back() method loads the previous URL in the windows.history list. It is the same as clicking the "back arrow" in your browser.
