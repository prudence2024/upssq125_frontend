Z-index Property

When elements are positioned they can overlap

The Z-INDEX property specify the stack order property of an element.

An element can have either a positive or negative stack order.

This is an iroko tree in Aminah village

Because the image has a z-index of -1 it will be placed behind the text, this shows that the higher the z-index the higher it will be in the stack and the lower te z-index, the lower it will be in the stack.
