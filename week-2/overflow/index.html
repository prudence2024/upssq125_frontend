Overflow Property Description

The overflow property specifies whether to click the content or to add scroll bars when the content of an element is too big to fit the specified area.

The overflow property has the following properties of which are visible, hidden, scroll, auto.

Note:The overflow property only work for block elements with a specified height.In oxslion(the operating system for mack computer) the scroll bars wll be hidden by default on this operating system and only show when being used even if the overflow is set on scroll.
