The Float Property

The float property is used for positioning and formatting content, for example when u want an image to float left in a container.

The float property can have one of the following values:

  • Left- The elements float to the left of its container.
  • Right- The elements float to the right of its container.
  • None- The elements does not float, it is the default property.
  • inherit- The element inherits the float value of its parent.

In its simplest use the float property is used to wrap text around images.
