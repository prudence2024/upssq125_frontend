What is asynchronous javascript

Asynchronous javscript allows your code to run without blocking the execution thread so that other task example UI update or network request cancontinue while waiting for somthing slow example fetching data from the server.

Key Concepts

Synchronous(blocking) Asynchronous(non blobking)
One task at a time. Task can run in the background
Slows down on long task. keeps the ui responsive
