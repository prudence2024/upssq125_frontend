Using a font that is easy to read is very important. The fonts add value to your text. It is also important to choose the correct color and text size for the font.

Generic font Families[Typeface]

In Css there are 5 generic font families:


  1. Serif:Serif fonts have a small stroke at the edges of each letter, they create the sense of formality and elegance
  2. Sans-Serif: These fonts have clean lines [no small strokes attached].They create a odern and minimalistic look.
  3. Monospace:These fonts have all their letters with the same fixed width,they create a mechanical look.
  4. Cursive:These fonts immitate human handwriting.
  5. Fantasy: These are decorative/playful fonts.

All the different font name belong to one of the generic name
