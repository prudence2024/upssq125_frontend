CSS Font-Family Property

In CSS we use the font-family property to specify the font of the text.

Note:If the font name is more than one word, it must be in quotation marks, like:"Times New Roman"

Tip:The Font-family property should hold several font name as a fall back system. The browser will try to use the first family in the list, if it is not available it will try the others.

Example


.p1{
font-family:"Times New Roman", Times , Serif ;
}
.p2{
font-family:Ariel, Helvetica , Sans-Serif;
}
.p3{
font-family:"Lucida console" "Courier New", Monospace;}
