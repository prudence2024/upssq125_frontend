HTML FORMS

An Html form is used to collect user input, the user input is most often sent to a server for processing.

The Form Element

An Html form element is used to create an html form for the user input and label

The form element is a container for different types on input element such as; text fills, checkboxes, radio buttons and submit button

The Input Element

The Input Element is the most used form element

Type Description
Input type text it displays the single line text fill
Input type radio it defines a radio button input fill that allows selection of one option from a set
Check box Defines a check box page that allows selection of multiple option from a set
Input Type Number Defines an input fill for entering a number
Input Type Email Defines an input fill for entering an email.
Input Type Password Defines an input fill for entering a password.
Upload This allows upload from a file from your computer.
