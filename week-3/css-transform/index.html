Transform

The transform property in CSS lets you rotate ,scale,skew or translate an element.

CSS 3d Transforms

CSS also support 3d Transformations

To use a 3d transform you need to set perspective property on the parent element

CSS 3d transform function

With the css 3d transform property, you can use the following transform function

  • perspective()
  • rotateX()
  • rotateY()
  • rotateZ()
  • translateX()
  • translateY()
  • translateZ()
  • scaleX()
  • scaleY()
  • scaleZ()
