The first splash into Javascript

We are gonna dive deep into javascript with a simple game that gives you an idea of how javascript features work and what writing Javascript feels like. Subsequently we will revisit al the features shown so don't worry if you don't understand all immediately.

Note: many of the code features you see in javascript are the same as in other programming languages - functions, loops etc. The code syntax looks differnt but the content are largely the same.


Number Guessing Game

We have selected a random number between 1 and 100 , see if you can guess it in 10 turns or fewer we will tell if your guess was too high or too low
