Javascript Operators

Javascript operators are SYNTAX used to perform different types of mathematical and logical computation. The '=' assign values, the "+" add values(value), the "*" multiplies values, the comparison " < >" compares values. The equal operator "==" is used for equating values. The '!=' is uesd for represnting not equal to. "===" is used for representing strict equal to(data type and value). the '-' sign represent arithematic subtraction. ** represent exponentation, '/' represent division,' %' represent modulus division. ++ represent increment by 1, -- represent decrement. x +=y x= x+y. &&- AND, || -or. Operand are the data or the representation of the data on which operators perform arithmetic and logical operation on.
