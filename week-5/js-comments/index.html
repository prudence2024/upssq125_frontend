Javascript Comments

As with html and css, it is possible to write comments into ur javascript code that will be ignored by the browser, comment provide instructions to ur fellow developers on how the code works(and you if you come back to ur code after six months and can't remember what you did). Comments are very useful and you should use them very often paticularly for large aplication .There are two type of comments:

  • "A single line comment is written after a double forward-slash("//"),e.g"

    JS 
                     
                    
                        // i am a comment

  • "A multi line comment is written between the strings"/* and * "example"

    /*i am also a comment*/
    Js 
                        
                        
                            //i am a comment
