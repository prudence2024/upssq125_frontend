They do the same thing for programming that ready mate furniture kit do for a ohome building. It is much easier to take ready cut panels and screw them together and make a book shelf rather than doing them yourself.

Browser Apis

  • (DOM)Document Object Model Api

  • The Geolocation Api

    The Geolocation Api retreives geographical information This is how google map is able to find your location and plot it on a map

  • The Canvas and WebGL Apis

    The Canvas and WebGL Apis allow u to create animated 2ds and 3ds graphics

  • Audio and Video Apis
