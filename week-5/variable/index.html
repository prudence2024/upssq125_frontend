Variables

Variables are containers for storing data, javascript variables can be declared in 4 ways:

  1. Automatically
  2. Using the var keyword.
  3. Using the Let keyword.
  4. Using the Const keyword.

note: it is good practice to declare your variable before using them

Using var: var x=y; var x=5; note: the var keyword was used in alll javascript code from 1995 till 2015, the let and const keyword where added to javascript in 2015. The var keyword should only be used for code written for older browsers.

Using let: let x=y; let x=5; only use let if you cant use const

Using const: const x=y; const x=5; use const if the value should not be changed, use it for an array or object

const price1 = 5; const price2 = 8; let total = price1 + price2;

where to use var, let or const. To declare a variable means to create the variable with the keyword

All javascript variables must be identified with unique names called identifiers.it can be short names like x and y or more descriptive names likes age, sum, total volume. Thee general rules for constructing variable names are:

  1. it can contain letters, digit, underscore and dollar sign
  2. it must begin with a letter.
  3. it can also begin with dollar sign and underscore but we will not use it in this tutorial.
  4. Names are case sensitive
  5. Reserved words like Javascript keywords cannot be used as a variable name.

The equal sign is an assignment operator, not an equal to, this is different from mathematics algebra. the equal to operator in mathematics is written as == in javscript.

Block scope

Before 2015, javscript didn't have block scope it only has globa and function scope. Scope in js refers to the accessibility or visibility of variables, in other words, where in your code the variable can be used.

  1. Global Scope: Any variable declared after . They can be accesed any where in your code in the same file or context . Overuse global variables can cause problems.
  2. Function Scope: Any variable declaredinside a function are only accessible within that function.
  3. Block Scope: Any variable declared inside let or const inside a block are only acccesible within that block. note: var doesnt have a blockscope.

Scope chain

When accessing a variable. javascript searches inside the current scope whether it is block or function, when it doesnt find the value it goes upward to the parent scope and it continue until it. You can't redeclared a variable with let or const
